# 🧮 CalcMaster – Enhanced Java Console Calculator

CalcMaster is a menu-driven Java console application that performs basic arithmetic operations, scientific calculations, and unit conversions. The project is developed using Object-Oriented Programming (OOP) concepts with a modular class-based structure.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- √ Square Root
- xʸ Power (Exponentiation)
- 🌡️ Temperature Conversion
  - Celsius → Fahrenheit
  - Fahrenheit → Celsius
- 💱 Currency Conversion
  - INR → USD
  - USD → INR
- ⚠️ Exception Handling
- 🔄 Continue/Exit Menu
- 📂 Modular Java Classes

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Scanner
- Exception Handling
- Java Math Library

## 🚀 How to Run

1. Clone the repository.
2. Open the project in Visual Studio Code or any Java IDE.
3. Compile all Java files.

```bash
javac *.java
```

4. Run the application.

```bash
java Main
```
